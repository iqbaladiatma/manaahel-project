Test Form - Tambah Media Cloudinary

Langkah-langkah:
1. Pastikan Anda sudah login sebagai Admin
2. Isi form di bawah dengan URL Cloudinary yang valid
3. Submit dan cek apakah muncul di gallery

Troubleshooting

  • Error 403: Pastikan login sebagai Admin
  • Error 422: Cek validasi form (URL harus valid)
  • Error 500: Cek log Laravel untuk detail error
  • Tidak muncul di gallery: Cek database apakah data tersimpan

Contoh URL Cloudinary yang Valid:

  • https://res.cloudinary.com/demo/image/upload/sample.jpg
  • https://res.cloudinary.com/demo/video/upload/sample.mp4
  • https://res.cloudinary.com/your-cloud/image/upload/v1234567890/folder/image.jpg
